# Введение в анализ
> [!Help]  Билет 1. Соответствие и Функция
>
> **Соответствие** – связь между элементами двух множеств X (область отправления) и Y (область прибытия).
>
> **Основные виды соответствий:**
> 1. Взаимно однозначное (каждому элементу X сопоставлен один Y).
> 2. Одно-многозначное (каждый X соответствует несколько Y).
> 3. Много однозначное (Каждому Y соответствует несколько X).
> 4. Много-многозначное (все друг другу соответствуют).
>
> **Функция** – это **функциональное** соответствие, где каждому элементу X сопоставлен ровно один элемент Y.
> *   `X` – область определения (D (f)).
> *   `Y` – область значений (E (f)).
> *   `f(x)` – значение функции в точке x.
>
> **Примеры функций:**
> 1. Линейная: `f(x) = 2x + 1`.
> 2. Квадратичная: `f(x) = x²`.
> 3. Показательная: `f(x) = 2ˣ`.
> 4. Логарифмическая: `f(x) = log₂(x)`.

> [!Help] Билет 2. Свойства числовых функций
>
> **Монотонность:**
> 1. **Возрастающая:** если `x₁ < x₂`, то `f(x₁) < f(x₂)`. *Пример:* `f(x) = 2x + 1`.
> 2. **Убывающая:** если `x₁ < x₂`, то `f(x₁) > f(x₂)`. *Пример:* `f(x) = -x + 3`.
>
> **Чётность:**
> 1. **Чётная:** `f(-x) = f(x)`. График симметричен относительно оси OY. *Пример:* `f(x) = x²`.
> 2. **Нечётная:** `f(-x) = -f(x)`. График симметричен относительно начала координат. *Пример:* `f(x) = x³`.
> 3. **Общего вида:** функция, не являющаяся чётной или нечётной. *Пример:* `f(x) = x + 1`.
>
> **Периодичность:**
> Существует число `T ≠ 0` (период) такое, что `f(x + T) = f(x)` для всех `x`.
> *Пример:* `f(x) = sin(x)`, `T = 2π`.

> [!Help] Билет 3. Ограниченность функции
>
> **Ограниченная функция:** Существует число C > 0, такое, что для всех x из области определения выполняется |f (x)| ≤ C. *Пример:* `f(x) = sin(x)`, где |sin (x)| ≤ 1.
>
> **Неограниченная функция:** Для любого числа C > 0 найдётся x, такое что |f (x)| > C. *Пример:* `f(x) = x²`.
>
> **Ограниченная сверху:** Существует число M, такое что f (x) ≤ M для всех x. *Пример:* `f(x) = -x²` (ограничена сверху числом 0).
>
> **Неограниченная сверху:** Для любого числа M найдётся x, такое что f (x) > M. *Пример:* `f(x) = x²`.
>
> **Ограниченная снизу:** Существует число m, такое что f (x) ≥ m для всех x. *Пример:* `f(x) = x²` (ограничена снизу числом 0).
>
> **Неограниченная снизу:** Для любого числа m найдётся x, такое что f (x) < m. *Пример:* `f(x) = -x²`.

> [!Help] Билет 4. Обратная функция
>
> **Механизм обращения:**
> 1. Исходное взаимно однозначное соответствие задано на `A × B` (как множество пар `(a, b)`).
> 2. Обратное соответствие задаётся на `B × A` как множество пар `(b, a)`, где `(a, b)` принадлежит исходному.
> 3. Пары меняются местами: `(аргумент, значение)` → `(значение, аргумент)`.
>
> **Взаимная обратимость:**
> Такие соответствия между `A` и `B` являются **взаимно обратными**. Если `f` – исходное отображение, то обратное обозначается `f⁻¹`. Обращение `f⁻¹` даёт снова `f`.
>
> **Для функции `y = f(x)`:**
> *   `A` – область определения `f`.
> *   `B` – область значений `f`.
> *   Пара – `(x, y)`.
> *   Обратная функция `f⁻¹(x)` задаёт пары `(y, x)`.

> [!Help] Билет 5. Классы функций f: Df ⊆ Rⁿ → Ef ⊆ Rᵐ
> 
> **1. F: Df ⊆ R¹ → Ef ⊆ R¹ (Скалярная функция скалярного аргумента)**
> *   **Характеристика:** Одна входная величина (скаляр) → одна выходная величина (скаляр).
> *   **Геометрическая интерпретация:** Кривая на плоскости (график `y = f(x)`).
> *   **Пример (физика):** `s(t) = v₀t + (at²)/2` — зависимость пути `s` от времени `t`.
>
> **2. F: Df ⊆ Rⁿ → Ef ⊆ R¹ (Скалярная функция векторного аргумента, n > 1)**
> *   **Характеристика:** Несколько входных величин (вектор `(x₁, ..., xₙ)`) → одно выходное значение (скаляр `y`).
> *   **Геометрическая интерпретация:** Для `n=2` — поверхность в трёхмерном пространстве (`z = f(x₁, x₂)`). Для `n>2` — гиперповерхность.
> *   **Пример (экономика):** `P(x₁, x₂) = p₁x₁ + p₂x₂ - C` — прибыль от производства `x₁` и `x₂` единиц двух товаров.
>
> **3. F: Df ⊆ R¹ → Ef ⊆ Rᵐ (Векторная функция скалярного аргумента, m > 1)**
> *   **Характеристика:** Одна входная величина (скаляр `t`) → несколько выходных величин (вектор `(y₁, ..., yₘ)`).
> *   **Геометрическая интерпретация:** Параметрически заданная кривая или траектория в пространстве Rᵐ.
> *   **Пример (физика):** `r(t) = (cos(t), sin(t), t)` — параметрическое задание винтовой линии в R³.
>
> **4. F: Df ⊆ Rⁿ → Ef ⊆ Rᵐ (Векторная функция векторного аргумента, n > 1, m > 1)**
> *   **Характеристика:** Несколько входных величин (вектор `(x₁, ..., xₙ)`) → несколько выходных величин (вектор `(y₁, ..., yₘ)`).
> *   **Геометрическая интерпретация:** Отображение (преобразование) одного многомерного пространства в другое.
> *   **Пример (компьютерная графика):** Аффинное преобразование `f(p) = A*p + b`, где `p ∈ R³`, `A` — матрица 3 x 3, `b` — вектор сдвига (преобразование 3 D-координат).

> [!Help] Билет 6. Композиция функций
>
> **Определение:** Композиция (сложная функция) `h = g ∘ f` — это функция, где результат `f` становится аргументом для `g`: `h(x) = g(f(x))`.
>
> **Область определения `D(g ∘ f)`:** Это все `x ∈ D(f)`, такие что `f(x) ∈ D(g)`.
>
> **Примеры:**
> 1. **R¹ → R¹:** `f(x)=x²`, `g(u)=sin(u)`, тогда `(g∘f)(x)=sin(x²)`.
> 2. **R² → R¹ → R¹:** `f(x,y)=x+y`, `g(v)=√v`, тогда `(g∘f)(x,y)=√(x+y)`.
> 3. **R¹ → R¹ → R³:** `f(t)=t²`, `g(s)=(s, s+1, cos(s))`, тогда `(g∘f)(t)=(t², t²+1, cos(t²))`.![[Pasted image 20251218232443.png]]
>    Чтобы найти область определения сложной функции, надо найти пересечение областей определения простых функций(подфункций).

> [!Help] Билет 7. Область определения, значения и график функции
>
> **Область определения (`Df ⊆ Rⁿ`)** — множество всех допустимых входных векторов `(x₁, ..., xₙ)`, для которых функция определена.
>
> **Область значений (`Ef ⊆ Rᵐ`)** — множество всех возможных выходных векторов `(y₁, ..., yₘ)`, которые получаются при подстановке всех `x ∈ Df`.
>
> **Примеры описания (n — размерность аргумента):**
> * **`n = 1`:** `Df` — промежуток на числовой прямой. *Пример:* `Df = (0, +∞)`.
> * **`n = 2`:** `Df` — область на плоскости. *Пример:* `Df = {(x,y) | x² + y² < 4}` (круг).
> * **`n = 3`:** `Df` — область в пространстве. *Пример:* `Df = {(x,y,z) | z ≥ 0}` (полупространство).
>
> **Примеры описания (m — размерность значения):**
> * **`m = 1`:** `Ef` — промежуток на числовой прямой. *Пример:* `Ef = [-1, 1]`.
> * **`m = 2`:** `Ef` — множество точек на плоскости. *Пример:* `Ef = {(u,v) | u = cos(t), v = sin(t)}` (окружность).
> * **`m = 3`:** `Ef` — множество точек в пространстве. *Пример:* `Ef = {(u,v,w) | w = 0}` (плоскость).
>
> **График функции `Γf`** — это множество упорядоченных пар вида `(x, f(x))` в пространстве `Rⁿ⁺ᵐ`, где `x ∈ Df`.
>
> **Когда имеет смысл говорить о графике?**
> Смысл визуализации и геометрической интерпретации есть, когда **`n + m ≤ 3`**:
> * **`n=1, m=1`:** График — линия на плоскости (`R²`). *Пример:* `y = x²`.
> * **`n=2, m=1`:** График — поверхность в пространстве (`R³`). *Пример:* `z = x² + y²`.
> * **`n=1, m=2`:** График — кривая в пространстве (`R³`). *Пример:* `(y,z) = (cos t, sin t)`.
> При `n+m > 3` график является абстрактным множеством в многомерном пространстве и не поддаётся прямой геометрической интерпретации.

> [!Help] Билет 8. Окрестности в R
>
> **Окрестность конечной точки `x₀`** — любой открытый интервал `(a, b)`, содержащий точку эту точку. Обозначается `U(x₀)` или `V(x₀)`.
> *Часто используется ε-окрестность:* `Uε(x₀) = (x₀ - ε, x₀ + ε)`, где `ε > 0`.
>
> **Односторонние окрестности:**
> * Правосторонняя: `U⁺(x₀) = [x₀, x₀ + δ)` или `(x₀, x₀ + δ)`.
> * Левосторонняя: `U⁻(x₀) = (x₀ - δ, x₀]` или `(x₀ - δ, x₀)`.
>  ![[Pasted image 20251219001301.png]]
>
> **Окрестности бесконечности:**
> * `U(+∞) = (M, +∞)`, где `M ∈ R`.
> * `U(-∞) = (-∞, M)`.
> * `U(∞) = (-∞, -N) ∪ (N, +∞)` (окрестность объединённой бесконечности).
>   ![[Pasted image 20251219001328.png]]
>
> **Способ задания:** Окрестности задаются с помощью интервалов и чисел (ε, δ, M, N), определяющих их размер или границу.

> [!Help] Билет 9. Точки и свойства множеств в Rⁿ
>
> **Внутренняя точка** множества `A` — точка, вокруг которой существует окрестность, целиком лежащая в `A`. *Пример:* Для `A = (0, 1)` точка `0.5` — внутренняя.
>
> **Граничная точка** множества `A` — точка, в любой окрестности которой есть как точки из `A`, так и точки не из `A`. *Пример:* Для `A = [0, 1)` точки `0` и `1` — граничные.
>
> **Граница множества (`∂A`)** — множество всех его граничных точек.
>
> **Открытое множество** — множество, все точки которого внутренние. *Пример:* Интервал `(0, 1)` или вся плоскость `R²`.
>
> **Замкнутое множество** — множество, содержащее все свои граничные точки. *Пример:* Отрезок `[0, 1]` или окружность вместе с ограниченным кругом.

# Дифференциальное исчисление и интегральное исчисление функций 

> [!Help] Билет 10. Производная
>
> **Производная** функции $f(x)$ в точке $x_0$ — предел отношения приращения функции к приращению аргумента при стремлении последнего к нулю:
> $f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$.
>
> **Механический смысл:** Мгновенная скорость. Если `s(t)` — закон движения, то `s'(t₀)` — мгновенная скорость тела в момент времени `t₀`.
>
> **Геометрический смысл:** Угловой коэффициент (тангенс угла наклона) касательной к графику функции в точке $(x_0, f(x_0))$.

> [!Help] Билет 11. Дифференциал функции
>
> **Дифференциал** функции $y = f(x)$ в точке $x_0$ — это главная линейная часть её приращения: $dy = f'(x_0) \cdot dx$, где $dx = \Delta x$ — произвольное приращение аргумента.
>
> **Пример:** Для $f(x) = x^2$, $f'(x) = 2x$. Дифференциал в точке $x_0=3$: $dy = 2 \cdot 3 \cdot dx = 6dx$. Если $dx = 0.1$, то $dy = 0.6$.
>
> **Применение в приближённых вычислениях:** При малых $dx$ приращение функции $\Delta y \approx dy$.
> **Формула:** $f(x_0 + \Delta x) \approx f(x_0) + f'(x_0) \cdot \Delta x$.
>
> **Пример вычисления:** $\sqrt{16.1} \approx \sqrt{16} + \frac{1}{2\sqrt{16}} \cdot 0.1 = 4 + \frac{0.1}{8} = 4.0125$.

> [!Help] Билет 12. Производные высших порядков
>
> **Производная $n$ -го порядка** функции $f(x)$ — производная от её производной $(n-1)$ -го порядка. Обозначается $f^{(n)}(x)$ или $\frac{d^n f}{dx^n}$.
>
> **Определение:** $f^{(n)}(x) = (f^{(n-1)}(x))'$, где $f^{(0)}(x) = f(x)$.
>
> **Примеры:**
> 1. $f(x) = x^4$
>     * $f'(x) = 4x^3$
>     * $f''(x) = 12x^2$
>     * $f'''(x) = 24x$
>     * $f^{(4)}(x) = 24$
>     * $f^{(5)}(x) = 0$
> 2. $f(x) = \sin(x)$
>     * $f'(x) = \cos(x)$
>     * $f''(x) = -\sin(x)$
>     * $f'''(x) = -\cos(x)$
>     * $f^{(4)}(x) = \sin(x)$ (далее циклически).

> [!Help] Билет 13. Дифференциальное уравнение
>
> **Дифференциальное уравнение (ДУ)** — уравнение, связывающее неизвестную функцию, её аргументы и её производные.
>
> **Решение ДУ** — функция, которая при подстановке в уравнение обращает его в тождество.
>
> **Пример:**
> * Уравнение: $\frac{dy}{dx} = 2x$.
> * Решение: $y = x^2 + C$, где $C$ — произвольная постоянная.
> * Проверка: $\frac{d}{dx}(x^2 + C) = 2x$.
